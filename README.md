# Intelli-Credit AI
### AI Powered Corporate Credit Decisioning Engine

Intelli-Credit AI is an intelligent system designed to automate corporate credit appraisal for banks and financial institutions. The system analyzes financial data, detects risks, and generates a Credit Appraisal Memo (CAM) using AI.

---

## Problem

Corporate loan evaluation is a slow and manual process. Credit managers must analyze multiple data sources such as:

- GST filings
- Bank statements
- Annual reports
- Legal records
- Industry news

This process often takes **weeks** and may miss important risk signals.

---

## Solution

Intelli-Credit AI automates the credit evaluation process using AI and data analytics.

The system:

- Extracts financial data
- Performs automated research
- Calculates credit risk
- Recommends loan amount
- Generates CAM reports

This reduces loan processing time from **weeks to minutes**.

---

## Features

- Multi-source data ingestion
- Financial risk scoring
- AI research agent
- Explainable credit scoring model
- Automated Credit Appraisal Memo generation
- Web dashboard for credit officers
